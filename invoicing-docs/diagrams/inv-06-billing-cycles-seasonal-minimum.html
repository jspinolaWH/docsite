PD-291 — Billing cycles

Each service or event type is linked to a predefined billing cycle (monthly, quarterly, annual). The system groups events automatically by their assigned schedule. Different services on the same customer account follow independent schedules.

PD-288 — Seasonal / time-based fees

Fees that exist independently of any operational event (base fees, monthly service fees, annual property charges, block collection fees, rents). The system auto-generates these as billable events on a configured schedule. No manual data entry required each period.

PD-286 — Minimum fee logic

PJH note: "If the previous owner has not met the minimum charge by November, the system adjusts the final invoice accordingly. If ownership changes during the year, the minimum charge adjustment must not apply to the new owner." Comparison uses VAT 0% (net) amount.
