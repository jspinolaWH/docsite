================================================================================
FIGMA DESIGN PROMPT — WasteHero 2.0 Product & Pricing Admin
Sprint 1 UI Spec (Release 1, March 2–13 2026)
================================================================================

PURPOSE
-------
This prompt defines the complete UI for Sprint 1 of the WasteHero 2.0 Product
& Pricing admin module. The output is a set of Figma frames covering:
  - All screens active in Sprint 1 (fully designed with real data)
  - The global app shell with ALL future navigation items visible but locked
  - The product detail tab shell with ALL future tabs visible but locked

Active Sprint 1 features: Product Categories, Waste Fractions, Pricing Units,
Zones (view-only hardcoded). Everything else shows a structured "coming soon"
empty state — not a dead end, but a clear placeholder.
